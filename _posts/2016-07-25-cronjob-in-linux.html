To create a cronjob on a linux-machine you just run

[code]
#The -e flag stands for edit
crontab -e

#The -l flag lists the cronjobs
crontab -l
[/code]

So I added my cronjob like this and program it to run every minute:

[code]
* * * * * /home/user/cron/script.sh
[/code]
